### impact-market-design-system

This is a design system based on Atomic Design pattern and styled-components. This package aim to host all UI elements used both on mobile and web applications developed by impact Market.

## Code organization

* eslint
* prettier
* typecheck

## Technologies

* typescript
* react-native
* React Navigation
* styled-components
